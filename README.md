# AI Writer Pro - AIライティングツール

Grok AIを搭載した次世代の日本語ライティングツールです。スマホからもPCからも快適に使えるレスポンシブデザインで、LLMによる文章生成と自分の執筆を同じエディターで同時に行えます。

## 🌐 アクセスURL

- **開発環境**: https://3000-i02zhtwkvt8b03kxgjjq0-2e1b9533.sandbox.novita.ai

## ✨ 実装済み機能

### 🔐 アカウント機能
- ユーザー登録・ログイン
- セッション管理（30日間有効）
- アカウント削除機能
- ユーザー設定（デフォルトモデル、テーマ、自動保存）

### 📝 ライティングエディター
- リアルタイム文字数カウント
  - 総文字数 / 空白除く文字数 / 行数
  - 読了時間（500文字/分で計算）
  - 原稿用紙換算（400字詰め）
- LLM生成テキストの直接編集
- 自動保存機能
- プロジェクト管理（保存・読み込み・削除）
- エクスポート機能（TXT/Markdown）

### 🎯 ジャンル選択
19種類のプリセットジャンル + 自由入力:
- 小説、エッセイ、ブログ、ビジネス文書、学術・論文
- 脚本・シナリオ、詩・俳句、ニュース記事、レビュー、SNS投稿
- メール、コピーライティング、技術文書
- ファンタジー、ミステリー、恋愛、ホラー、SF
- その他（カスタム入力）

### 💡 ネタ考案機能
- テーマ・キーワード入力
- 追加条件の設定
- アイデア数の選択（3/5/10個）
- 生成結果の保存

### 📊 プロット作成機能
- アイデアからプロット生成
- 詳細度選択（シンプル/標準/詳細）
- プロットから執筆モードへの移行

### 🤖 AI執筆サポート
- **続きを書く**: 現在の文章の続きを自動生成
- **書き直す**: 選択部分をより良い表現に
- **拡張する**: 選択部分を詳しく描写
- **目標文字数指定**: 生成する文字数を指定可能

### 🔧 編集ツール
- **校正機能**: 誤字脱字・文法チェック
- **要約機能**: 長文を簡潔に要約
- **タイトル案生成**: 内容に最適なタイトルを5つ提案

### 🎨 文体変換
- **敬語変換**: ビジネス文体に
- **カジュアル変換**: 口語体に
- **文学的変換**: 美しい表現に

### 🔧 Grokモデル選択
- Grok 3 (最新) - 最も高性能
- Grok 3 Fast - 高速応答版
- Grok 2 - バランス型
- Grok 2 (1212) - 安定版

### 📱 レスポンシブデザイン
- スマホ・タブレット最適化
- モバイル用AIアシスタントパネル
- ダークモード対応

### 📜 生成履歴
- AI生成の全履歴を保存
- 履歴からの再利用・コピー
- 生成タイプ別フィルタリング

## 🏗️ 技術スタック

- **データベース**: Cloudflare D1 (SQLite)

## 📝 使い方

1. **アカウント作成**: メールアドレス、パスワード、ユーザー名で登録
2. **プロジェクト作成**: 「+」ボタンでジャンルとタイプを選択
3. **執筆**: エディターに文章を入力
4. **AI活用**: 右パネルのAIアシスタントで生成・編集
5. **保存**: 自動保存 or 手動保存ボタン
6. **エクスポート**: TXT/Markdown形式でダウンロード
